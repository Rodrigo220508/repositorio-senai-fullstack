- O que é uma estrutura de repetição?

**R:  Uma estrutura de repetição é um recurso da
programação que permite executar um mesmo bloco de código várias vezes, de acordo
com uma condição ou uma quantidade definida de repetições.**


- Para que serve o laço? 

**R: O laço for é usado quando se sabe antecipadamente quantas
vezes uma ação deve ser repetida. Ele reúne em uma única estrutura a inicialização, a
condição de repetição e a atualização da variável de controle.**


- Para que serve o laço while?

**R:  O laço while é usado quando não se sabe exatamente
quantas vezes a repetição ocorreu. Ele executa o bloco de código enquanto uma condição
for verdadeira.**


- Para que serve o laço do-while?

**R:  O laço do-while também executa um bloco de
código enquanto uma condição for verdadeira, mas garante que o bloco seja executado
pelo menos uma vez, pois a condição é verificada apenas após a primeira execução.**


- Qual é a diferença entre for, while e do-while?

**R: O for é usado quando se sabe o
número de repetições. O while repete enquanto uma condição for verdadeira. O
do-while também repete por condição, mas executa o código pelo menos uma vez.**
