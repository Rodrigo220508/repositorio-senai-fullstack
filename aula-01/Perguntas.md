# Perguntas

- O que é programação?

**R: É o processo de criar instruções para que um computador execute tarefas.**


- O que é um algoritmo?

**R: É uma sequência de passos para resolver um problema ou realiza uma tarefa.**


- Qual a diferença entre algoritmo e linguagem de programaçao?

**R: Algoritmo é a lógica,  enquanto a linguagem de programação é a ferramenta usada para  escrever essa lógica.**


- O que é JavaScript?

**R: É uma linguagem de programação.**

- Onde o JavaScript pode ser utilizado atualmente?

**R: Desenvolvimento front-end, back-end, aplicativos mobile  e sistemas desktop.**

- O que é o Node.js?

**R: é um ambiente de execução de código aberto e multiplataforma que permite aos desenvolvedores executar JavaScript fora do navegador web.**

- Qual é a função do Node.js no desenvolvimento com JavaScript?

**R: A função permite executar código JavaScript fora do navegador, transformando-o em uma linguagem de uso geral capaz de criar desde servidores web e robustos até ferramnetas de automação e aplicativos de linha de comando.**

- O que é o VS Code?

**R: É um editor de código-fonte gratuito e de código aberto desenvolvido pela Microsoft.**

- Qual a diferença entre um editor de texto comum e um editor de código?

**R:Editores de texto comuns servem apenas para escrever e salvar texto puro. Editores de código são ferramentas especializadas para programadores, projetadas especificamente para aumentar a produtividade, detectar erros e facilitar a escrita de software.**

- O que é o terminal?

**R: Um ponto de conexão ou extremidade.**

- Para que serve o comando node arquivo.js?

**R: Para executar um código JavaScript diretamente no seu computador ou servidor, fora de um navegador web.**

- O que é uma variável?

**R: É um elemento ou símbolo que representa um valor ou característica que pode mudar ou ser alterado.**

- Qual a diferença entre let e const?

**R: Capacidade de reatribuição.**

- O que são tipos de dados? Cite exemplos.

**R:Classificação que indicam ao seu computador qual é a natureza, o tamanho e quaisoperções podem ser feitas com um determinado dado (texto, número, verdadeiro/falso).**

- Qual a principal diferença entre JavaScript e TypeScript?

**R:TypeScript é uma  linguagem tipada estatícamente, enquanto o JavaScript é dinamicamente tipado.**
