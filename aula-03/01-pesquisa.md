# Pesquisa - Estruturas Condicionais

- O que é uma estrutura condicional em programação? Dê um exemplo do cotidiano usando "se... então... senão".

**R: É uma estrutura que permite ao programa tomar decisões. Exemplo: Se estiver chovendo, então levo guarda-chuva, senão saio sem ele.**

- Qual é a diferença entre usar apenas if e usar if/else?

**R: O if executa uma ação apenas quando a condição é verdadeira. O if/else permite executar uma ação quando a condição é verdadeira e outra quando é falsa.**

- Quando usamos else if? Crie um exemplo com três ou mais possibilidades.

**R: Usamos else if quando existem várias condições para testar. Exemplo: Se a nota for maior ou igual a 9, Excelente. Senão, se for maior ou igual a 6, Aprovado. Senão, Reprovado.**

- O que significa uma condição retornar true ou false?

**R: Significa que a condição foi avaliada. True indica que ela é verdadeira e false indica que ela é falsa.**

- Qual é a diferença entre == e === em JavaScript? Qual deles é mais recomendado?

**R: O == compara apenas os valores. O === compara valor e tipo. O mais recomendado é o === porque evita erros de comparação.**

- Para que servem os operadores &&, || e !? Explique com um exemplo para cada um.

**R: O && significa "E". Exemplo: idade >= 18 && temCadastro.
O || significa "OU". Exemplo: temCupom || ehAluno.
O ! significa "NÃO". Exemplo: !bloqueado.**

- O que o operador % faz? Por que ele é útil para descobrir se um número é par ou ímpar?

**R: O operador % retorna o resto da divisão. Se numero % 2 == 0, o número é par. Caso contrário, é ímpar.**

- O que é switch case? Em que situação ele fica melhor do que vários else if?

**R: É uma estrutura usada para escolher entre várias opções. Fica melhor quando estamos comparando uma mesma variável com vários valores diferentes.**

- Para que servem break e default dentro de um switch?

**R: O break encerra o caso atual do switch. O default é executado quando nenhum dos casos corresponde ao valor informado.**

- Por que é importante testar o mesmo programa com vários valores diferentes?

**R: Porque ajuda a encontrar erros e garante que o programa funcione corretamente em diferentes situações.**
